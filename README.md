# 열역학 경로 최적화 시뮬레이터 v2.0

이상기체의 상태 변화에서 **최적 경로**를 찾고, 다양한 **열역학 사이클**을 시뮬레이션하는 대화형 웹 애플리케이션입니다.

## 주요 기능

### 1. 경로 분석
- **등온/등압/등적/단열** 과정 시뮬레이션
- **Dijkstra** 및 **A*** 알고리즘으로 최적 경로 탐색
- **P-V 다이어그램** 시각화
- 열역학적 성질 계산 (W, Q, ΔU, ΔH, ΔS, ΔG)

### 2. 열역학 사이클
- **오토 사이클** (가솔린 엔진)
- **디젤 사이클** (디젤 엔진)
- **브레이턴 사이클** (가스 터빈)
- **카르노 사이클** (이론적 최대 효율)

### 3. 3D 시각화
- **P-V-T 다이어그램** (Plotly 인터랙티브)
- 이상기체 상태방정식 표면 시각화

### 4. 알고리즘 비교
- Dijkstra vs A* 성능 비교
- 다양한 휴리스틱 함수 (맨해튼, 유클리드, 열역학적)

## 실행 방법

### 1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 앱 실행

```bash
streamlit run app.py
```

브라우저가 자동으로 열리며 시뮬레이터가 실행됩니다.

## 파일 구조

```
thermodynamics-path-optimizer/
├── app.py              # Streamlit 메인 앱
├── thermodynamics.py   # 열역학 계산 모듈
├── pathfinding.py      # 경로 탐색 알고리즘 (Dijkstra, A*)
├── visualization.py    # 시각화 모듈 (2D, 3D)
├── examples.py         # 예제 시나리오
├── requirements.txt    # 의존성 목록
└── README.md           # 문서
```

- pandas >= 2.0.0
- scipy >= 1.10.0

## 사용 방법

1. **기체 타입 선택**: 단원자/이원자/다원자 기체
2. **상태 설정**: 초기(A) 및 최종(B) 상태의 P, V 설정
3. **경로 추가**: 등온/등압/등적/단열 중 선택
4. **최적 경로 찾기**: Dijkstra 또는 A* 알고리즘 선택
5. **결과 분석**: 그래프 및 테이블로 확인, CSV 다운로드

## 탭 설명

| 탭 | 설명 |
|---|------|
| 경로 분석 | P-V 다이어그램과 경로별 비교 |
| 열역학 사이클 | 오토/디젤/브레이턴/카르노 사이클 시뮬레이션 |
| 3D 시각화 | P-V-T 공간에서 경로 시각화 |
| 알고리즘 비교 | Dijkstra vs A* 성능 비교 |

## 열역학 상수

| 상수 | 값 | 설명 |
|------|-----|------|
| R | 0.0821 L·atm/(mol·K) | 이상기체 상수 |
| γ (단원자) | 5/3 | 비열비 (He, Ne, Ar) |
| γ (이원자) | 7/5 | 비열비 (N₂, O₂, H₂) |
| γ (다원자) | 4/3 | 비열비 (CO₂, H₂O) |

## v2.0 새로운 기능

- 기체 타입 선택 (단원자/이원자/다원자)
- A* 알고리즘 추가 (열역학적 휴리스틱)
- 3D P-V-T 다이어그램 (Plotly)
- 열역학 사이클 시뮬레이션 (오토, 디젤, 브레이턴, 카르노)
- 엔탈피(ΔH), 깁스 자유에너지(ΔG) 계산
- 다크모드 지원
- CSV 결과 내보내기
- 알고리즘 성능 비교
